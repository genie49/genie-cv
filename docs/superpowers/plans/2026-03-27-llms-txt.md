# llms.txt Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** AI가 `https://genie-cv.com/llms.txt`를 통해 포트폴리오 전체 정보를 구조적으로 읽을 수 있도록, 빌드 시 자동 생성되는 llms.txt + 마크다운 파일을 만든다.

**Architecture:** `scripts/generate-llms.ts` 스크립트가 `data/*.json` + `data/content/**/*.md`를 읽어서 `apps/client/public/`에 llms.txt 및 개별 마크다운 파일을 생성한다. Vite 빌드 시 `dist/`로 복사되어 Railway에서 정적 서빙된다.

**Tech Stack:** Bun (runtime, fs API), TypeScript

**Branch:** `feature/llms-txt` (from `develop`)

**Spec:** `docs/superpowers/specs/2026-03-27-llms-txt-design.md`

스크립트의 기본 구조와 `profile.md` 생성 로직을 작성한다.

- [ ] **Step 1: 스크립트 기본 구조 작성**

`scripts/generate-llms.ts`:
```typescript
import { readFileSync, writeFileSync, mkdirSync, rmSync, existsSync, copyFileSync, readdirSync } from "fs";
import { join } from "path";

const BASE_URL = "https://genie-cv.com";
const ROOT = join(import.meta.dir, "..");
const DATA = join(ROOT, "data");
const OUTPUT = join(ROOT, "apps/client/public");

// 출력 디렉터리 초기화
const llmsDir = join(OUTPUT, "llms");
if (existsSync(llmsDir)) rmSync(llmsDir, { recursive: true });
mkdirSync(join(llmsDir, "projects"), { recursive: true });
mkdirSync(join(llmsDir, "notes"), { recursive: true });

// JSON 데이터 로드
const profile = JSON.parse(readFileSync(join(DATA, "profile.json"), "utf-8"));
const projects = JSON.parse(readFileSync(join(DATA, "projects.json"), "utf-8"));
const qna = JSON.parse(readFileSync(join(DATA, "qna.json"), "utf-8"));

// --- 생성 함수들 ---

function generateProfile(): string {
  const { name, role, email, github, about, techStack, experience, education } = profile;

  let md = `# ${name} | ${role}\n\n`;
  md += `- Email: ${email}\n`;
  md += `- GitHub: ${github}\n\n`;
  md += `## 소개\n\n${about}\n\n`;

  // 기술 스택
  md += `## 기술 스택\n\n`;
  for (const [category, skills] of Object.entries(techStack)) {
    md += `### ${category}\n`;
    md += `- ${(skills as string[]).join(", ")}\n\n`;
  }

  // 경력
  md += `## 경력\n\n`;
  for (const exp of experience) {
    md += `### ${exp.company} — ${exp.title} (${exp.period})\n`;
    md += `${exp.description}\n\n`;
  }

  // 학력
  md += `## 학력\n\n`;
  for (const edu of education) {
    md += `### ${edu.school} — ${edu.major}\n`;
    md += `- 학위: ${edu.degree}\n`;
    md += `- 기간: ${edu.period}\n`;
    md += `- 상태: ${edu.status}\n\n`;
  }

  return md;
}

writeFileSync(join(llmsDir, "profile.md"), generateProfile());
console.log("✓ profile.md");
```

function generateProjectDetail(project: any): string {
  let md = `# ${project.title}\n\n`;
  md += `> ${project.description.split("\n")[0]}\n\n`;
  md += `- 기간: ${project.period}\n`;
  md += `- 태그: ${project.tags.join(", ")}\n`;
  if (project.demo) md += `- 데모: ${project.demo}\n`;
  md += `\n`;

  // 주요 기능
  if (project.features?.length) {
    md += `## 주요 기능\n\n`;
    for (const feat of project.features) {
      md += `- **${feat.title}**: ${feat.description}\n`;
    }
    md += `\n`;
  }

  // content/projects/{slug}.md 병합
  const contentPath = join(DATA, "content/projects", `${project.slug}.md`);
  if (existsSync(contentPath)) {
    const content = readFileSync(contentPath, "utf-8");
    md += `## 상세\n\n${content}\n\n`;
  }

  // 관련 노트 링크
  if (project.notes?.length) {
    md += `## 관련 기술 노트\n\n`;
    for (const note of project.notes) {
      md += `- [${note.title}](${BASE_URL}/llms/notes/${note.id}.md)\n`;
    }
    md += `\n`;
  }

  return md;
}

for (const project of projects) {
  writeFileSync(join(llmsDir, "projects", `${project.slug}.md`), generateProjectDetail(project));
  console.log(`✓ projects/${project.slug}.md`);
}
```

function generateLlmsTxt(): string {
  const { name, role } = profile;
  let txt = `# ${name} | ${role}\n\n`;
  txt += `> AI 엔지니어 ${name}의 포트폴리오. 프로필, 프로젝트, 기술 노트, Q&A를 담고 있다.\n\n`;

  // Profile
  txt += `## Profile\n`;
  txt += `- [프로필](${BASE_URL}/llms/profile.md): 소개, 기술 스택, 경력, 학력\n\n`;

  // Projects
  txt += `## Projects\n`;
  txt += `- [프로젝트 목록](${BASE_URL}/llms/projects.md): 전체 프로젝트 요약\n`;
  for (const project of projects) {
    txt += `- [${project.title}](${BASE_URL}/llms/projects/${project.slug}.md): ${project.description.split("\n")[0]}\n`;
  }
  txt += `\n`;

  // Q&A
  txt += `## Q&A\n`;
  txt += `- [Q&A](${BASE_URL}/llms/qna.md): 자기소개, 개발 스타일, 협업 방식 등 7개 문답\n\n`;

  // Notes
  txt += `## Technical Notes\n`;
  // projects.json의 notes에서 제목 가져오기
  const noteMeta = new Map<string, string>();
  for (const project of projects) {
    for (const note of project.notes || []) {
      noteMeta.set(note.id, note.title);
    }
  }
  // 모든 노트 파일 나열 (메타데이터 있으면 제목 사용, 없으면 파일명)
  for (const file of noteFiles) {
    const id = file.replace(".md", "");
    const title = noteMeta.get(id) || id;
    txt += `- [${title}](${BASE_URL}/llms/notes/${file})\n`;
  }

  return txt;
}

writeFileSync(join(OUTPUT, "llms.txt"), generateLlmsTxt());
console.log("✓ llms.txt");

console.log("\n완료!");
```

> 주의: `noteFiles` 변수가 Task 4에서 정의되므로, llms.txt 생성은 노트 복사 이후에 실행되어야 한다.
