# 김형진 | AI Engineer

AI와 소프트웨어 엔지니어링을 결합하여 실용적인 제품을 만드는 것에 관심이 많은 개발자입니다.

## 핵심 역량

- **LLM 애플리케이션 개발**: LangChain, LangGraph를 활용한 에이전트 시스템 설계 및 구현
- **RAG 파이프라인**: 벡터 데이터베이스(LanceDB)와 임베딩 모델을 활용한 검색 증강 생성 시스템 구축
- **풀스택 웹 개발**: React, ElysiaJS, FastAPI를 활용한 프론트엔드/백엔드 개발
- **실시간 시스템**: SSE, WebSocket을 활용한 실시간 데이터 스트리밍 구현

## 관심 분야

AI 엔지니어링, LLM 애플리케이션, RAG 시스템, 풀스택 개발, 실시간 데이터 처리
