# 김형진 | AI Engineer

한번 몰입하면 빠르게 결과물을 만들어내는 개발자입니다. AI 기술이 실제 비즈니스 가치를 만드는 과정에 강한 흥미를 가지고 있으며, 딥러닝 모델부터 LLM 기반 Multi-Agent 시스템, 풀스택 웹 서비스까지 아이디어를 프로덕트로 완성하는 전 과정을 경험했습니다.

## 핵심 역량

### 멀티에이전트 시스템 설계

여러 프로젝트에서 복잡한 워크플로우를 에이전트 협업 구조로 풀어왔습니다. fingoo에서는 Supervisor 패턴으로 7개 전문 에이전트를 오케스트레이션하고, kimpro에서는 Account Manager가 5개 에이전트에게 4단계 마케팅 캠페인을 위임하는 구조를 설계했습니다. 단일 에이전트의 한계를 넘어, 에이전트 간 상태 공유와 실패 복구까지 고려한 프로덕션 레벨 시스템을 구축한 경험이 있습니다.

### 컨텍스트 엔지니어링 & RAG 파이프라인

에이전트가 올바른 판단을 하려면 적절한 컨텍스트가 필수입니다. fingoo에서는 pgvector + Tavily 웹 검색을 결합한 2단계 RAG를, bonda에서는 PDF 파싱 → OCR 폴백 → 듀얼 벡터 이미지 검색까지 포함하는 End-to-End RAG 파이프라인을 설계했습니다. 텍스트뿐 아니라 CSV, PDF, 이미지 등 멀티모달 데이터를 AI가 활용할 수 있도록 변환하는 파이프라인 경험을 갖추고 있습니다.

### Agent 품질 평가 체계

프롬프트를 수정할 때마다 다른 곳이 망가지는 문제를 해결하기 위해, fingoo에서 MockService 기반 평가 환경과 6가지 Grader로 에이전트 품질을 수치화하는 평가 하네스를 구축했습니다. 평가 결과를 기반으로 프롬프트를 자동 개선하는 피드백 루프를 만들어, 에이전트 품질을 지속적으로 관리할 수 있는 체계를 갖추었습니다.

### 비즈니스 문제를 Agent 워크플로우로 전환

kimpro에서 마케팅팀의 반복 업무(캠페인 기획 → 브리프 → 매칭 → 가이드)를 분석하고, 각 단계를 Agent 노드로 변환하여 4단계 자동화 워크플로우를 구축했습니다. 기술 중심이 아니라 비즈니스 병목을 먼저 파악하고, 그에 맞는 Agent 구조를 설계하는 접근 방식을 지향합니다.

### 풀스택 E2E 구현력

프론트엔드(React, Next.js)부터 백엔드(FastAPI, Fastify, ElysiaJS), AI(LangChain), 모바일(Kotlin), 인프라(Docker, AWS, GCP)까지 혼자 또는 소규모 팀으로 아이디어를 프로덕트로 완성해왔습니다. 특히 Agent 시스템에 필수적인 실시간 통신(WebSocket, SSE)과 스트리밍 파이프라인 구현 경험을 갖추고 있습니다.
