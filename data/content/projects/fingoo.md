# 핀구 — AI 투자 분석 플랫폼

AI 대화형 채팅으로 금융 지표 분석·차트 시각화·경제 전망을 제공하는 올인원 투자 분석 플랫폼.

## 프로젝트 개요

개인 투자자를 위한 AI 기반 투자 분석 서비스입니다. LangChain 멀티 에이전트가 DART 공시, yfinance 해외 주가, Tavily 웹 검색 등 다양한 데이터 소스를 종합하여 실시간 스트리밍 채팅으로 투자 인사이트를 제공합니다. 인터랙티브 금융 차트, 통계 분석 도구, 게이미피케이션 기반 투자 교육까지 포함하는 종합 플랫폼입니다.

## 기술 스택

- **프론트엔드**: Next.js 14 + TypeScript + Socket.io + Zustand + Mantine
- **AI 서비스**: FastAPI + LangChain (Python) — Grok·Claude·Gemini 멀티 모델
- **백엔드**: NestJS + TypeORM + PostgreSQL + Redis
- **인프라**: Docker + GitHub Actions + AWS EC2 + Nginx + Fluent Bit
- **데이터**: DART API, yfinance, FRED, Tavily, pgvector

## 주요 기능

### AI 대화형 채팅

LangChain Supervisor 패턴으로 7개 전문 에이전트(시장분석·기술분석·리서치·포트폴리오·퀀트·시각화·지표)를 오케스트레이션합니다. 총 81개의 도구를 활용하며, 사용자가 자연어로 요청하면 Tool Calling을 통해 차트 생성, 지표 예측, 화면 분할 등 프론트엔드 UI를 직접 조작합니다.

**주요 분석 역량:**
- **포트폴리오 분석**: 수익률, 샤프 비율, 소르티노 비율, MDD, VaR, 베타, 상관관계 매트릭스
- **기술적 분석**: 이동평균, RSI, MACD, 볼린저밴드, OBV, 지지/저항선, 차트/캔들 패턴 감지
- **퀀트 분석**: 옵션 가격(Black-Scholes), 옵션 그리스, 내재 변동성, 채권 가격, 수익률 곡선, 듀레이션
- **기업 리서치**: DART 재무제표, 배당 이력, 수익 추정치, 자사주 매입 정보
- **차트 시각화 생성**: AI가 트리맵, 라인차트, 캔들스틱, 바차트, 파이차트를 직접 생성

**멀티모달 입력**: CSV, XLSX, PDF, 이미지, HWP 등 다양한 파일을 업로드하면 AI가 내용을 이해하고 분석에 활용합니다.

**미들웨어 스택**: 10개의 미들웨어(중단 감지, 도구 호출 정규화, 재귀 방지, 동적 모델 선택, LLM 폴백, 인용 강제, 서브에이전트 위임, 미완료 작업 추적, 도구 호출 제한, 컨텍스트 요약)로 에이전트 행동을 세밀하게 제어합니다.

### 실시간 차트

주식·암호화폐·채권·원자재·ETF 등 멀티 에셋을 지원하는 차트 시스템입니다.

- **레이아웃**: 풀스크린/2분할/4분할 뷰, 다중 Y축 지원
- **인터랙션**: 추세선·수평선 드로잉, 드래그 앤 드롭 지표 배치, 날짜 범위 탐색
- **통계 분석 도구 (Beta)**: 회귀분석, GARCH, RSI 등 내장 통계 지표
- **커스텀 예측 지표**: 사용자가 직접 예측 지표를 생성하고 차트에 오버레이
- **내보내기**: CSV 데이터 내보내기, 차트 이미지 다운로드 및 공유

외부 금융 데이터 API와 연동하여 실시간 시세를 조회하고, 8개 주요 시장 지수(S&P 500, NASDAQ, KOSPI, KOSDAQ 등)를 제공합니다.

### 시황 RAG

매일 경제 데이터를 자동 수집하여 시황을 생성하고, 이를 pgvector 기반 RAG 시스템으로 구축합니다. AI 채팅에서 경제 이슈에 대해 질문하면 수집된 시황 데이터를 컨텍스트로 활용하며, 불충분할 경우 Tavily 웹 검색으로 보완하는 2단계 파이프라인입니다.

### 투자 교육 & 게이미피케이션

4챕터 구조의 단계별 투자 학습 모듈입니다.

- **학습 시스템**: 챕터별 학습 → 퀴즈(객관식, 숫자 블록, 배열) 플로우
- **게이미피케이션**: XP 진행 바, 게임 코인, 일일 출석 체크(연속 출석 보상), 시즌제 랭킹 시스템
- **보상 체계**: 퀴즈 완료 시 코인 보상, 트로피 시스템

### Human-in-the-Loop 차트 제어

AI가 차트를 직접 수정할 때 Unix의 read/write 권한 모델에서 착안한 조회/변경 분리 패턴을 적용합니다. 조회 도구(심볼 검색, 차트 데이터 확인)는 즉시 실행하고, 변경 도구(지표 추가/삭제, 기간 변경 등 7개)는 interrupt를 발생시켜 사용자 확인을 거칩니다. PostgreSQL 체크포인터로 에이전트 상태를 영속화하여 서버 재시작에도 대화가 보존됩니다.

## 담당 영역

창업팀에서 FE/AI 개발자로 참여하여 다음을 담당했습니다:

- **AI 채팅 프론트엔드**: Socket.io 기반 실시간 스트리밍 채팅 UI, 시각화 커맨드 버퍼링, 도구 실행 상태 추적, interrupt 확인 UI
- **AI 에이전트 설계**: Supervisor-Subagent 아키텍처 설계, 미들웨어 스택, Human-in-the-Loop 패턴
- **시황 RAG 파이프라인**: 경제 데이터 크롤링 → pgvector 임베딩 → Tavily 웹 검색 보완
- **에이전트 평가 시스템**: MockService 기반 평가 환경, 6가지 Grader로 품질 측정
- **CI/CD 인프라**: GitHub Actions + Docker + AWS EC2 배포 파이프라인, SSM 환경변수 관리
- **로깅 시스템**: Wide Event 철학 기반 구조화 로깅, FluentBit → Discord 알림
