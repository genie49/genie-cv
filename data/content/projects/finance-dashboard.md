# 실시간 금융 대시보드

WebSocket 기반 실시간 주식/암호화폐 데이터를 시각화하는 대시보드.

## 프로젝트 개요

실시간 시세 데이터를 WebSocket으로 수신하고, 인터랙티브 차트로 시각화하는 풀스택 금융 대시보드입니다. 포트폴리오 분석과 가격 알림 기능을 제공합니다.

## 기술 스택

- **프론트엔드**: Next.js + TypeScript
- **백엔드**: FastAPI + WebSocket
- **데이터베이스**: PostgreSQL
- **차트**: Recharts

## 주요 기능

### 실시간 데이터 스트리밍
WebSocket을 통해 실시간 시세 데이터를 수신하고 차트에 즉시 반영합니다.

### 포트폴리오 분석
보유 자산 기반 수익률 계산 및 리스크 분석 대시보드를 제공합니다.

### 알림 시스템
사용자가 설정한 가격 조건에 따라 이메일/슬랙 알림을 발송합니다.
